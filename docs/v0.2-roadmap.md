# v0.2 Roadmap

## Deferred Features

### Context Managers
- [ ] `temporary_policy()` - Override rcParams temporarily
- [ ] `theme_context()` - Switch themes (mocha/latte/frappe/macchiato)
- [ ] `backend_context()` - Backend switching utilities
- [ ] Font registration/caching system

### Decorators
- [ ] `@validate_data` - Shape/type checking for plot inputs
- [ ] `@auto_labels` - Derive labels from pandas/xarray metadata
- [ ] `@publication_ready` - Enforce quality constraints (DPI, fonts, etc)
- [ ] `@save_on_exit` - Auto-save decorator with path template

### Additional Primitives
- [ ] `line()` - Line plots with error bands
- [ ] `bar()` - Bar charts (vertical/horizontal)
- [ ] `histogram()` - Distribution plots
- [ ] `heatmap()` - 2D density/correlation matrices
- [ ] `subplot_grid()` - Multi-panel layouts

### Policy Enhancements
- [ ] Typography scale system (modular scale or golden ratio)
- [ ] Aspect ratio policies (publication presets: square, golden, 16:9)
- [ ] DPI policies (screen/print/poster presets)
- [ ] Multiple Catppuccin flavor support (not just mocha)
- [ ] Colorblind-safe palette validation

### Examples
- [ ] More example scripts (line, bar, histogram)
- [ ] Jupyter notebook with comparisons

### Advanced Features
- [ ] Pandas DataFrame integration
- [ ] Xarray DataArray support
- [ ] LaTeX label support
- [ ] Animation utilities
- [ ] Interactive backend helpers

## Won't Do (Out of Scope)
- 3D plotting
- Geographic/map plots
- Network graphs
- Custom backends
